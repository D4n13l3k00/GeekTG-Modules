# GeekTG Modules

Модули для юзербота [GeekTG](https://github.com/D4n13l3k00/GeekTG) — форка Friendly Telegram.

## Совместимость

> [!WARNING]
> Модули **не совместимы** с оригинальным [Friendly Telegram](https://github.com/friendly-telegram/friendly-telegram) и форком GeekTG, на основе которого сделан [D4n13l3k00/GeekTG](https://github.com/D4n13l3k00/GeekTG).
> Используйте только с указанным форком.

## Установка модуля

В чате с юзерботом:

```
.dlmod <ссылка на raw файл модуля>
```

## Разработка

Окружение на базе [uv](https://github.com/astral-sh/uv):

```bash
uv sync --dev
```

Требуется Python 3.13+.
